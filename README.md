# Computer Architecture Lab 5 

#### Program options
| Long form | Short form | Arguments |
|:---|:---:|---:|
| -\-greeting | -g | no args |
| -\-num | -n | decimal |
| -\-bye | -b | no args |
| -\-help | -h | no args |

#### Make options
| target | description |
|---|---|
| **compile** | compile cpp file into ***```target```*** folder |
| **run** | run file (if compiled) with ***```-h```*** argument |
| **clean** | clean tager directory |
